# 🏆 QuizMaster Pro

QuizMaster Pro is a high-performance, visually stunning React-based quiz application designed for elite knowledge testing. It features a sleek glassmorphism UI, real-time analytics, and dynamic lifelines to enhance the user experience.

![QuizMaster Dashboard](https://images.unsplash.com/photo-1606326608606-aa0b62935f2b?q=80&w=2070&auto=format&fit=crop)

## ✨ Features

- **🚀 Performance-First**: Built with Vite for lightning-fast HMR and build times.
- **💎 Premium Design**: Fully responsive glassmorphism UI powered by Vanilla CSS and Framer Motion.
- **🧠 Expert Engine**: Custom quiz hook managing timers, lifelines (50:50, Skip), and scoring logic.
- **📊 Real-time Analytics**: Detailed performance reports with accuracy tracking and category stats.
- **🥇 Global Leaderboard**: Persistent ranking system tracking the top players.
- **🔐 Secure Auth**: Local session management for personalized progress tracking.

## 🛠️ Technology Stack

- **Framework**: React 19
- **Build Tool**: Vite 8
- **Styling**: Vanilla CSS (Custom Design System)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Effects**: Canvas Confetti

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Rohanpandey914064/react-learning.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 📖 How to Play
1. **Initialize Access**: Enter your alias to start your session.
2. **Choose a Category**: Select from Science, Technology, or Math challenges.
3. **Beat the Clock**: You have 15 seconds per question.
4. **Use Lifelines**:
   - **Zap (50:50)**: Removes two incorrect options.
   - **Skip**: Move to the next question without penalty.
5. **Analyze Results**: Review your detailed performance and climb the Global Hall of Fame.
